Overview

ion.lua is a simple JSON inspired compact data storage format designed for Lua purposes. This module's name is stylised as "ion", in all lowercase.

Prerequisites

To use this module, you will need at least Lua 5.2+ or LuaJIT 2.0+. This module is (well, obviously...) incompatible with Luau.
